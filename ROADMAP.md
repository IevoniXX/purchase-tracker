# Roadmap

Future vision for PurchaseTracker, roughly in priority order.

## Phase 1: Make it usable (current)

- [x] View purchases by status (wish list, ordered, pickup, delivered)
- [x] Filter by family member
- [x] Add new items
- [x] Delete items with confirmation
- [ ] Edit existing items (change status, update delivery dates, add notes)
- [ ] Move items through lifecycle with one click (e.g., "Mark as delivered")

## Phase 2: Make it shareable

- [ ] Deploy to Railway/Render/Fly.io (see DEPLOYMENT.md)
- [ ] Add simple login (two accounts: Ieva, Cesar) — no registration needed, just password-protected
- [ ] Mobile-friendly layout (currently works on mobile but could be tighter)

## Phase 3: Make it smarter

- [ ] Notifications / reminders
  - "IKEA shelf arriving tomorrow — someone needs to be home"
  - "Bosch drill at post office — 2 days left to pick up"
- [ ] Budget tracking — monthly spending limit, progress bar
- [ ] Price comparison notes — "cheapest on Amazon at €45, but €38 on eBay used"
- [ ] Shared wish list voting — both can upvote/downvote wish list items

## Phase 4: Home inventory (future extension)

Track everything you own, not just recent purchases.

- [ ] Move delivered items into inventory automatically
- [ ] Add items you already own (not purchased through the app)
- [ ] Track: age, condition, original cost, estimated resale value
- [ ] Categories: electronics, furniture, kitchen, tools, kids, etc.
- [ ] "Time to replace" alerts based on age/condition
- [ ] "Sell or toss" view — items you no longer need, with suggested resale price
- [ ] Export inventory for insurance purposes
